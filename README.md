# Лабораторная работа №4

## Вариант: Библиотека + Казино и Гуси

* **Цель:** Реализовать систему управления библиотекой и игровую симуляцию казино с использованием пользовательских коллекций, наследования и псевдослучайной симуляции.
* **Библиотеки:** *random*, *pytest*
* **Реализовано:**
  - Система управления библиотекой с классами Book, Magazine, Library
  - Пользовательская списковая коллекция BookCollection
  - Пользовательская словарная коллекция IndexDict для индексации по ISBN, автору и году
  - Игровая симуляция казино с классами Player, Goose, WarGoose, HonkGoose, Casino
  - Пользовательские коллекции GooseCollection, PlayerCollection, ChipBalanceDict
  - Псевдослучайная симуляция для обоих вариантов
  - Полное покрытие тестами с использованием pytest

### Алгоритм решения
1. **Библиотека:**
   - Класс Item — базовый класс для книг и журанлов
   - Классы Book и Magazine — наследники Item
   - BookCollection — пользовательская списковая коллекция
   - IndexDict — пользовательская словарная коллекция для индексации
   - Library — основной класс, управляющий коллекциями
   - Симуляция включает события: добавление и удаление книги, поиск, взятие и возврат книг

2. **Казино и Гуси:**
   - Класс Goose — базовый класс для гусей
   - Классы WarGoose и HonkGoose — наследники Goose
   - Класс Player — игрок с балансом фишек
   - GooseCollection и PlayerCollection — пользовательские списковые коллекции
   - ChipBalanceDict — пользовательская словарная коллекция для отслеживания балансов
   - Casino — основной класс, управляющий игровой симуляцией
   - Симуляция включает события: добавление игроков/гусей, ставки, атаки, крики, панику

## Инструкция к использованию

### Запуск симуляций
Убрать во всех импортах точки и запустить код через терминал
```bash
cd src
python main.py
cd ..
```
### Запуск тестов
Добавить во все импорты точки и запустить через терминал
```bash
pip install pytest
cd tests
pytest test.py
cd ..
```
